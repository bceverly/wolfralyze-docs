# Wolfralyze Website

This repository contains the GitHub Pages website for [Wolfralyze](https://wolfralyze.org) - Professional Code Quality Analysis for Wolfram Mathematica™.

## About Wolfralyze

Wolfralyze is a Tier 1 SonarQube® plugin that brings enterprise-grade code quality analysis to Wolfram Mathematica™. With 529+ analysis rules, it provides the same depth of analysis available for mainstream languages like Java and Python.

## Website

The website is hosted at: **https://wolfralyze.org**

## Local Development

To view the website locally:

```bash
# Clone the repository
git clone https://github.com/bceverly/wolfralyze-docs.git
cd wolfralyze-docs

# Open index.html in your browser
open index.html
```
